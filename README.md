# Frontend

React + HeroUI панель для Bot CRM.

## Запуск

1. Установи зависимости:
   `npm install`
2. Скопируй `.env.example` в `.env` и при необходимости поменяй `VITE_API_URL`
3. Запусти:
   `npm run dev`

## Вход

Для первого входа используй seed-аккаунт из backend:

- `admin@local.dev`
- `admin12345`
